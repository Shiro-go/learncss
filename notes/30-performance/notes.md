# CSS Performance

## 1. Concept

CSS performance affects how fast your page renders.

---

## 2. Key Factors

- selector complexity
- repaint/reflow
- file size

---

## 3. Best Practices

- use simple selectors
- avoid deep nesting
- minimize unused CSS

---

## 4. Expensive Operations

- layout changes
- large DOM updates
- heavy animations

---

## 5. Optimization Tips

- use `transform` instead of top/left
- avoid animating layout properties
- reduce CSS size

---

## 6. Mental Model

Performance = less work for the browser

---

## 7. Common Mistakes

- overly complex selectors
- unnecessary animations
- huge stylesheets

---

## 8. Mini Exercise

Optimize a layout using fewer selectors.

---

## 9. Summary

Efficient CSS improves rendering speed and UX.
