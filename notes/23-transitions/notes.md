# Transitions

## 1. Concept

Transitions create smooth changes between states.

---

## 2. Example

```css
button {
  transition: background 0.3s;
}

button:hover {
  background: red;
}
```

---

## 3. Properties

- duration
- timing function
- delay

---

## 4. Mental Model

Transition = smooth interpolation

---

## 5. Common Mistakes

- forgetting property to animate
- too long duration
- animating non-animatable properties
