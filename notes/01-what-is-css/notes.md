# What is CSS

## 1. Concept

CSS (Cascading Style Sheets) is used to control the visual appearance of HTML elements.

HTML = structure  
CSS = style  

Without CSS, websites are plain and unstyled. :contentReference[oaicite:0]{index=0}

## 2. Why it exists

CSS separates content from design:
- HTML → content
- CSS → presentation

This makes code cleaner and easier to maintain.

## 3. What CSS controls

- Colors
- Fonts
- Layout
- Spacing
- Animations

## 4. Example

```css
p {
  color: red;
}
```

## 5. Mental Model

"Select elements → apply styles"

## 6. Common Mistakes

- Using only HTML for styling
- Overusing inline CSS
- Ignoring separation of concerns

## 7. Mini Exercise

Create a paragraph and:
- make it blue

## 8. Summary

CSS defines how your website looks and separates design from structure.
