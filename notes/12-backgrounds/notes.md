# Backgrounds

## 1. Concept

Background properties control the visual layer behind elements.

---

## 2. Background Color

```css
div {
  background-color: black;
}
```

---

## 3. Background Image

```css
div {
  background-image: url("image.jpg");
}
```

---

## 4. Background Control

```css
div {
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;
}
```

---

## 5. Shorthand

```css
div {
  background: black url("img.jpg") no-repeat center/cover;
}
```

---

## 6. Mental Model

Background = visual layer behind content

---

## 7. Common Mistakes

- Image not covering correctly
- Repeating backgrounds unintentionally
- Ignoring responsiveness
