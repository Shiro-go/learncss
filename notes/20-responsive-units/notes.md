# Responsive Units

## 1. Concept

Responsive units adapt to screen or parent size.

---

## 2. Units

- % → relative to parent
- rem → root font size
- em → parent font size
- vw → viewport width
- vh → viewport height

---

## 3. Example

```css
div {
  width: 50%;
  height: 100vh;
  font-size: 2rem;
}
```

---

## 4. Why It Matters

Fixed units break layouts on different screens.

---

## 5. Mental Model

Responsive units = flexible scaling

---

## 6. Common Mistakes

- using px everywhere
- mixing units poorly
- misunderstanding em

---

## 7. Mini Exercise

Create a fully responsive section using vw/vh.

---

## 8. Summary

Responsive units make layouts adaptable.
