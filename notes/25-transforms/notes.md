# Transforms

## 1. Concept

Transforms modify element shape or position.

---

## 2. Examples

```css
div {
  transform: translateX(50px);
  transform: rotate(45deg);
  transform: scale(1.2);
}
```

---

## 3. Combine

```css
div {
  transform: translateX(50px) rotate(10deg);
}
```

---

## 4. Mental Model

Transforms = visual manipulation without affecting layout

---

## 5. Common Mistakes

- overriding transform unintentionally
- not combining transforms
- confusing layout vs transform

---

## 6. Mini Exercise

Rotate and scale an element on hover.

---

## 7. Summary

Transforms allow powerful visual effects without breaking layout.
