# Borders & Shadows

## 1. Concept

Borders define edges.  
Shadows create depth.

---

## 2. Borders

```css
div {
  border: 2px solid black;
}
```

You can control:
- width
- style
- color :contentReference[oaicite:0]{index=0}

---

## 3. Border Radius

```css
div {
  border-radius: 10px;
}
```

Rounded corners.

---

## 4. Box Shadow

```css
div {
  box-shadow: 0 4px 10px rgba(0,0,0,0.3);
}
```

---

## 5. Text Shadow

```css
h1 {
  text-shadow: 2px 2px 5px black;
}
```

---

## 6. Mental Model

- border = structure  
- shadow = depth

---

## 7. Common Mistakes

- Overusing shadows
- Too strong blur
- Inconsistent borders

## 9. Summary
