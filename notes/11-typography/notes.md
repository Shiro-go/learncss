# Typography

## 1. Concept

Typography controls how text looks and is displayed.

Good typography = readability + design quality.

---

## 2. Core Properties

### Font family

```css
body {
  font-family: Arial, sans-serif;
}
```

### Font size

```css
p {
  font-size: 16px;
}
```

### Font weight

```css
p {
  font-weight: bold;
}
```

---

## 3. Text Styling

```css
p {
  text-align: center;
  line-height: 1.5;
  letter-spacing: 1px;
}
```

---

## 4. Web Fonts

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto');
```

---

## 5. Mental Model

Typography = readability first, style second

---

## 6. Common Mistakes

- Using too many fonts
- Bad line spacing
- Hard-to-read font sizes

---

## 7. Mini Exercise

Style a paragraph:
- change font
- adjust spacing
- center text

---

## 8. Summary

Typography directly impacts user experience and readability.
