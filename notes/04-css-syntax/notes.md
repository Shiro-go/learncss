# CSS Syntax

## 1. Concept

CSS is made of rules.

Each rule = selector + declarations

## 2. Structure

```css
selector {
  property: value;
}
```

Example:

```css
p {
  color: red;
  font-size: 16px;
}
```

## 3. Parts

- selector → what to style
- property → what to change
- value → how to change

## 4. Multiple Declarations

```css
p {
  color: red;
  background: black;
}
```

## 5. Rules

- End each line with `;`
- Use `{}` for blocks
- Properties must be valid

## 6. Mental Model

CSS = instructions applied to elements

## 7. Common Mistakes

- Missing `;`
- Typos in properties
- Invalid values

## 8. Mini Exercise

Write CSS that:
- changes color
- adds background
- changes size

## 9. Summary

CSS syntax is simple but must be precise.
