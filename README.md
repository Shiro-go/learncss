# CSS Roadmap — Structured Learning Project

A structured and visual roadmap to learn CSS from fundamentals to advanced concepts.

This project is designed as both:
- a **learning system**
- a **documentation platform**

Each concept is explained through dedicated notes and organized into sections that reflect real-world CSS progression.

---

## 📚 Overview

This roadmap covers:

- Fundamentals (syntax, selectors, cascade)
- Styling (typography, colors, backgrounds)
- Layout (box model, flexbox, grid)
- Responsive design (media queries, units)
- Advanced CSS (animations, transforms, variables)
- Professional practices (architecture, performance, accessibility)

The goal is to **understand CSS deeply**, not just memorize properties.

---

## 🧱 Project Structure

```
.
├── index.html          # Main entry (roadmap overview)
├── style.css           # Global styling (glass / glow UI)
├── script.js           # Animations and interactions
├── sections/           # Section pages (grouped topics)
├── notes/              # Detailed lessons (markdown)
└── README.md
```

### Notes system

Each concept has its own folder:

```
notes/
└── 14-flexbox/
    └── notes.md
```

Each `notes.md` is a **complete lesson**, not just quick notes.

## 🧠 Learning Philosophy

This project is built around:

- **Progressive learning** → from basics to advanced
- **Separation of concerns**
  - UI → HTML/CSS/JS
  - Knowledge → Markdown notes
- **Scalability** → can extend to JS, backend, etc.
